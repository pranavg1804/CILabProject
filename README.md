CI Lab Project



This project demonstrates Continuous Integration using Jenkins.



Tools Used:

\- Jenkins

\- GitHub

\- Maven

\- Java

\- JUnit



Implemented Features:

\- Freestyle Jenkins Job

\- Multibranch Pipeline Job

\- Automated Build \& Test

\- GitHub Integration

\- Jenkinsfile based pipeline



Project Structure:

\- src/        : Source code

\- scripts/    : Build and deploy scripts

\- docker/     : Dockerfile

\- Jenkinsfile : Pipeline definition



How to Run:

1\. Clone repository

2\. Run scripts/build.bat

3\. Observe Jenkins build
